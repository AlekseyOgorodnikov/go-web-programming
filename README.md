## WEB PROGRAMMING GO

`go version 1.17.3`

### Подключение к БД Postgres супер пользователь Linux

`sudo su postgres`

### Создание нового пользователя

`createuser --interactive <username>`

### Вывод списка фалгов ролей пользователя.

`createuser --help`

### Создать базу данных

`createdb <database name>`
